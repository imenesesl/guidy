# Landing

Public-facing landing page for Guidy.

## Getting Started

### Prerequisites

- Node.js 22 (see `.nvmrc`)
- pnpm

### Install

```bash
pnpm install
```

### Run

```bash
pnpm dev:landing
```

Dev server starts on `http://localhost:3001`.

### Test

```bash
pnpm --filter @guidy/landing test
```

### Storybook

```bash
pnpm storybook:landing
```

Storybook runs on port 6008.
